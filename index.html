Proposito del sitio

En el siguiente proyecto de investigacion se procedera a realizar el desarrollo de un sitio web, el cual estara encargado de mostrar cada uno de los temas que seran tratados a lo largo del curso, de tal manera que se puede brindar un contexto adecuado que permita entender cuales seran los procesos a tratar en este trabajo en particular, cabe recalcar que el objetivo de este tipo de proyectos es fortalecer los conocimientos basicos que se poseen en cuanto al diseño de sitios web, ya sea para trabajos, estudio, o incluso para crear una pagina propia, en donde se puedan realizar arreglos de una manera personalizada, y sin la necesidad de pagar a otra persona por realizar esta tarea, durante el desarrollo de este proyecto se realizara un investigacion enfocada en el uso de la sintaxis y selectores del lenguaje css, ademas de la interaccion humano-ordenador que se ejecuta cuando una persona esta interactuando con algun sitio web en especifico.

Sintaxis y los selectores en CSS

Los selectores CSS se utilizan para seleccionar elementos HTML en función de su nombre de elemento, de la identificación, clase, atributo, y mucho más. Mientras que, la sintaxis es un conjunto de reglas CSS consta de un selector y un bloque de declaración, esto se puede observar en la figura que se encuentra al lado de este texto.

Interacción humano computador

La interacción humano computador es la disciplina que estudia cómo las personas interactúan con las computadoras y hasta qué punto las computadoras se desarrollan para interactuar con las personas.
